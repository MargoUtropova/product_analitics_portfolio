## EDA анализ и рекомендательная система на основе датасета фильмов Netflix, взятого с сайта kaggle.

**Проект состоит из четырех частей:**
* в первой посмотрим на сами данные и какие есть закономерности
* во второй построим рекомендательную систему по признаку "похожести" фильмов на основе описания фильмов, которое мы преобразуем в токены и применим к ним построение матрицы на основании косинусного расстояния.
* в третьей части визуализируем проект в виде удобного пользователю интерфейса через сервис streamlit
* далее подумаем, какие еще возможны способы рекомендаций или фильтров и как их встроить в сервис.

*Второй раздел уже доступен к просмотру.*

### Навигация по папкам:
* dataset -- ссылка на скачивание датасета
* funcs -- функции вынесены в отдельные файлы для удобства
* insights -- папка для систематизации выводов
* notebooks -- здесь файлы .ipynb с рабочим кодом
* visuals -- сохраненные графики из блокнотов и скриншоты с сайта после запуска проекта
