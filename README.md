# My Data Analytics Portfolio

Привет! 👋  
Это мое портфолио учебных и pet-проектов по анализу данных.  
Здесь я собираю кейсы, в которых работаю с данными: от гипотез и очистки до выводов и визуализаций.

---

## 🗂 Структура репозитория
Для удобства просмотра папки организованы тематически: один проект - одна тема. Внутри общей папки содержатся папки с материалами по проекту и файл `readme.md` с пояснением, что находится внутри папок и о чем проект.

| Папка | Что внутри |
|-------|------------|
| datasets | Исходные данные для проектов (CSV, Excel, ссылки на источники). Если файлы слишком большие — лежит описание или ссылка на скачивание. |
| notebooks | Jupyter Notebooks с кодом на Python: исследование данных, очистка, расчет метрик, проверка гипотез. |
| visuals | Готовые графики, скриншоты дашбордов, визуализации. Всё, что помогает «увидеть» данные. |
|insights | Выводы по каждому проекту: что нашли, какие гипотезы подтвердились/опроверглись, какие рекомендации можно дать. |
